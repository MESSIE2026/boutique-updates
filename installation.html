Installation — MESSIEMATALA POS

1) Ce qu’il faut

  • Windows 10/11
  • .NET Framework 4.8 (si nécessaire)
  • SQL Server Express (recommandé)

2) Installer

  1. Télécharger le setup depuis la page principale
  2. Lancer Setup.exe et terminer l’installation
  3. Ouvrir l’application

3) Première configuration

  • Serveur : .\SQLEXPRESS
  • Base : BoutiqueDB_NomBoutique
  • Créer base → Enregistrer → Relancer l’application
